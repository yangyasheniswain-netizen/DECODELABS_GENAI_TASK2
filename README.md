# Automated Copywriting & Tone Transformer

## Overview

This project is an AI-powered Python application that generates marketing content based on the product name, platform, and writing tone. It uses the Groq API to create engaging promotional text.

## Features

* AI-generated marketing content
* Supports different writing tones
* Easy to use
* Fast content generation

## Technologies Used

* Python
* Groq API
* Llama 3.3 70B Model
* VS Code

## How to Run

1. Install the Groq library:

```bash
pip install groq
```

2. Add your Groq API key in `chatbot.py`.

3. Run the program:

```bash
python chatbot.py
```

* Product: Smart Watch
* Platform: Instagram
* Tone: Professional
